# STRIDE Threat Analysis

| STRIDE Category | Component | Threat |
|---|---|---|
| Spoofing | Authentication Server | Attacker impersonates identity provider |
| Tampering | Backend API | Malicious API request modifies sensitive records |
| Repudiation | Web Application | User performs action without audit trail |
| Information Disclosure | Database | Sensitive user data exposed |
| Denial of Service | Web Application | Traffic flood causes downtime |
| Elevation of Privilege | Authentication Server | Normal user gains admin privileges |
